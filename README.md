# gdg-6th-ai-mission

안녕하세요!  
GDG Hongik Univ. AI 파트를 담당하고 있는 김수진, 정승윤입니다.

## 미션 코스 소개
미션 코스는 프로젝트가 시작되는 방학 전에, 4주간 프로젝트에 필요한 기초적인 실력을 키우기 위해 만들어졌어요.  
실질적인 AI 프로젝트 수행을 위해, 데이터 처리부터 모델 학습, 그리고 배포까지 핵심 개념을 단계적으로 학습하는 과정입니다.


## 문제 해결 방법
미션 진행 중 막히는 부분이 있다면 아래 방법을 활용해보세요:

- 문제 내용을 복사하여 검색 및 관련 블로그 자료 탐색
- AI를 활용하여 질문
- 팀원들과 문제 상황 공유 및 토론
- 디스코드 프로젝트 트랙 서버에 캡쳐와 함께 질문 업로드

## 제츨 기한
- **월요일 시작** / **일요일 제출** 원칙

## WIL (Weekly I Learned)
AI 미션 코스는 다른 파트에 비해 **WIL(Weekly I Learned) 작성에 더 많은 시간이 소요**될 수 있습니다.  
WIL까지 함께 작성해야 하므로, 충분한 시간을 고려하여 계획적으로 진행해 주세요.
